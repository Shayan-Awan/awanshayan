NeuroSpectrum: A Brainwave-Powered Vision Enhancement Device

Description

  • Shayan Awan

  • https://bit.ly/45dJRNL

  • Aug 2023 - Current

ANeuroSpectrum uses novel predictive modelling to monitor visual stress in the visual cortex using Electroencephalography (EEG) technology. The data is filtered through various frequencies and assessed in order to swap to the according filter to reduce symptoms. An object detection model was implemented in order to identify the task which the patient was pursuing, this data is then saved with the according filter in the environment. The device is an affordable, light weight device equipped with bluetooth and voice recognition sensors to aid patients in easily using my device for reducing symptoms caused by certain visual perceptions.
